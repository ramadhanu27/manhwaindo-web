# Cara Mendapatkan Sitelinks di Google Search

## Apa itu Sitelinks?
Sitelinks adalah link tambahan yang muncul di bawah hasil pencarian utama di Google, menampilkan halaman-halaman penting dari website Anda. Contoh: seperti detik.com yang menampilkan "detikNews", "detikSport", "Terpopuler", dll.

**BreadcrumbList Schema:**
- Struktur navigasi website
- Membantu Google memahami hierarki halaman

### 2. Sitemap
Sudah ada di `public/sitemap.xml` (auto-generated)
- Berisi semua halaman penting
- Update otomatis setiap build

## ğŸ” Cara Mempercepat Munculnya Sitelinks

### 1. Optimasi Header/Navigation
Pastikan link penting ada di header:
```tsx
<nav>
  <Link href="/">Home</Link>
  <Link href="/series">Daftar Manhwa</Link>
  <Link href="/search">Pencarian</Link>
  <Link href="/bookmark">Bookmark</Link>
</nav>
```

### 2. Optimasi Footer
Tambahkan section "Quick Links" di footer dengan link ke halaman penting.

### 3. Buat Halaman "Sitemap HTML"
Selain XML sitemap, buat halaman HTML yang list semua halaman penting.

### 4. Internal Linking
Dari homepage, link ke halaman-halaman penting dengan anchor text yang jelas.

### 5. Google Search Console
- Submit sitemap
- Request indexing untuk halaman penting
- Monitor performance
